> **本章核心**: 掌握高速电路中最常用四种元器件的选型和应用要点，通过真实故障案例理解错误选型带来的后果。

## 2.1 电阻的应用

### 📌 经典案例

#### 【案例 2-1】串联电阻过大导致板间告警失败

**电路描述**:

业务板上的电源监控芯片检测到电源异常时，通过告警信号线(ERR)将低电平信号经由背板传递到主控板。主控板上的 74LVTH16244 逻辑芯片接收该信号，处理后触发 CPU 中断。

由于单板支持热插拔，告警信号与背板连接器之间串有电阻: 业务板侧 R1=1kΩ，主控板侧 R2=100Ω、R3=1kΩ。

![图2.1 业务板与主控板之间的告警电路](images/Ch02_fig2.1_alarm_circuit.png)

**现象**: 强制将业务板被监控电源拉地，CPU 中断信号不使能。

**根因分析**:

74LVTH16244 为高阻抗输入，当监控芯片输出低电平(0V)时，主控板输入端的电平由 R3 与 R1+R2 分压决定:

$V_{in} = 3.3V \times \frac{R_1+R_2}{R_1+R_2+R_3} = 3.3V \times \frac{1k\Omega + 100\Omega}{1k\Omega + 100\Omega + 1k\Omega} = 3.3V \times \frac{1.1k\Omega}{2.1k\Omega} \approx 1.73V$

1.73V 远高于 74LVTH16244 的低电平输入门限(典型值 0.8V)，所以 CPU 无法识别告警。

**解决**:

将 R1 从 1kΩ 改为 33Ω:

$V_{in} = 3.3V \times \frac{33\Omega + 100\Omega}{33\Omega + 100\Omega + 1k\Omega} = 3.3V \times \frac{0.133k\Omega}{2.133k\Omega} \approx 0.38V$

0.38V 处于低电平门限(≤0.8V)内，告警功能恢复正常。

> 注意: 不能通过增大 R3 来进一步降低电平，因为逻辑器件实现电平翻转时有最小电流要求，R3 过大可能导致输入电流不足(详见第3章)。

**教训**: 多板协同设计时，每块单板单独看都没有问题，但接口组合后产生分压效应。设计背板接口电路必须考虑本板与其他单板的协同工作。

#### 【案例 2-2】电阻额定功率不够造成的单板潜在缺陷

**电路描述**:

PHY 芯片(以太网物理层芯片)的内核电源(VCC_CORE)对噪声特别敏感，设计中使用 LC 滤波后再串联一个 1Ω 电阻 R，用做全频段衰减器(不仅能滤高频噪声，也能衰减低频噪声)。

![图2.2 PHY芯片电源滤波电路](images/Ch02_fig2.2_PHY_filter.png)

**现象**: 单板长时间运行后，电阻 R 频繁爆裂。

**根因分析**:

- 电阻 R 采用 0402 封装，额定功率 1/16W
- 该电阻的最大通流能力: $I_{max} = \sqrt{P/R} = \sqrt{0.0625W / 1\Omega} = 0.25A$
- PHY 芯片内核最大功耗 300mW @ 1.5V，意味着最大电流:

$I = P / V = 300mW / 1.5V = 200mA$

即 PHY 全速工作时，电阻上的实际功率为:

$P = I^2 \times R = (0.2A)^2 \times 1\Omega = 0.04W$

虽然 0.04W < 0.0625W(额定功率)看起来满足，但 200mA > 62.5mA(额定电流的估算),而且 0402 电阻的散热能力差，长时间运行热量累积导致失效。

实际上，0402 封装 1/16W 电阻按 62.5mA 最大电流计算已远超，PHY 内核 200mA 电流是安全的 3 倍以上。

**关于品质因数 Q 的讨论**:

这个电路中的电阻 R 除了全频段衰减外，还有一个重要作用——**降低电路的品质因数 Q**。

$Q$ 的定义: 回路谐振时，储存能量与一周期内消耗能量之比。对于一个 RLC 串联电路:

$Q = \omega_0 L / R = 1 / (\omega_0 C R) = (1/R) \times \sqrt{L / C}$

不同 Q 值的幅频特性如下图所示:

![图2.3 不同Q值的幅频特性曲线](images/Ch02_fig2.3_Q_curve.png)

从上图可见:
- **Q 值大**: 能量集中于谐振频率附近，曲线陡峭，损耗小。在选频电路(如滤波器)中希望 Q 大，选频特性好，滤除其他频带信号的能力越强；
- **Q 值小**: 能量分散，曲线平缓。在电源或信号线路中希望 Q 小，避免谐振引起的振铃和失真

本案例中，串联电阻 R 降低了 RLC 回路的 Q 值，防止电源线路因谐振产生振荡。如果只用 LC 滤波而不串电阻，LC 储能元件可能产生自激振荡，使电源噪声反而增大。

**解决**: 更换为额定功率足够的电阻(如 0603 或更大封装)，同时兼顾阻值精度。

**教训**: 
1. 电阻选型不仅要关注阻值，额定功率(额定电流)至少降额 20%
2. 串联在电源路径上的电阻还起到降低 Q 值、防止谐振的作用
3. 电阻的失效可能不会立即暴露，需要通过长时间老化测试发现

#### 【案例 2-3】0Ω 电阻在时序设计中的妙用

**电路描述**:

FPGA 需要兼容两个厂家的存储器(A 和 B)，两者引脚定义完全相同，但时序参数有差异。经时序计算，B 厂家的时钟信号线需要比 A 厂家长 600mil 才能满足建立/保持时间。

![图2.4 FPGA兼容两种存储器的设计](images/Ch02_fig2.4_FPGA_compat.png)

**解决方案**: 使用 0Ω 电阻 + PCB 走线实现兼容:

- 使用 **A 厂家存储器**时: 焊接 R1，R2/R3 不上件。时钟信号经 R1 直接到达 A 存储器
- 使用 **B 厂家存储器**时: 不焊 R1，焊接 R2 和 R3。时钟信号经 R2 → 600mil 走线 → R3 到达 B 存储器，利用 600mil 走线长度补偿时序差

**PCB 设计要点**:

1. R2 需紧靠 R1 左侧引脚放置，R3 紧靠 R1 右侧引脚放置——减少时钟信号线上可能出现的分叉(stub)
2. R2 与 R3 之间的 PCB 走线长度为 600mil

对于信号速率极高、不允许存在分叉的设计，可以采用更先进的方式:

![图2.5 无分叉的兼容性设计方法](images/Ch02_fig2.5_no_stub.png)

PCB 上将 R2 和 R3 各自的一个引脚焊盘与 R1 的两个引脚焊盘分别重合，R2 和 R3 的另一个引脚通过 600mil 走线相连。这种方式完全避免了分叉，但会产生 DRC(设计规则检查)错误，需手动忽略并通知产品工程师。

**教训**: 0Ω 电阻搭配 PCB 走线是实现多方案兼容的简便手段，但信号速率高时要注意分叉对信号完整性的影响，必要时采用焊盘重合的设计。

### 📌 电阻应用要点

高速电路中电阻选型需关注**四个维度**:

| 维度 | 要点 |
|------|------|
| **阻值** | 尽量用常用阻值串并联构成非常用值，避免非标件采购困难且成本高 |
| **尺寸** | EIA 英制代码(如 0603 表示 0.06in×0.03in)与公制代码(1608 表示 1.6mm×0.8mm)，注意区分 |
| **额定功率** | **至少降额 20%**，大电流路径上的电阻必须重点核算。0402 ≈ 1/16W，0603 ≈ 1/10W，0805 ≈ 1/8W |
| **精度** | 电源分压电阻等设定参数的场合必须选 1% 精度，否则输出电压偏差过大 |

**精度应用示例**:

![图2.6 LT3080 利用外部电阻设定输出电压](images/Ch02_fig2.6_LT3080.png)

LT3080 的输出电压由外部电阻 $R_{SET}$ 设定。若选 5% 精度的电阻，输出电压波动可达 10%，必须选 1% 精度电阻。即使 1% 精度，仅电阻偏差就导致了 2% 的输出电压偏差。

> **降额原则**: \ $ P_{实际} \leq P_{额定} \times 80\% $ ，即实际功率不得超过额定功率的 80%

## 2.2 电容的选型及应用

### 经典案例

#### 【案例 2-4】电容失效导致低温下硬盘停止工作

**现象**: -30°C 低温冷启动时，硬盘检测报错。

**根因**: 硬盘 5V 电源滤波使用 47μF 铝电解电容，标称温度范围虽达 -55°C~105°C，但低温下 ESR 急剧增大(约为常温的 2 倍以上)，滤波效果大幅下降，导致 5V 纹波达 800mV。

**解决**: 将铝电解电容更换为同容值的钽电容，低温测试通过。

**教训**: 铝电解电容不宜用于低温环境或对电源质量要求高的场景。

#### 【案例 2-5】多次带电插拔子板导致母板上钽电容损坏

**现象**: 子板热插拔十多次后母板上 12V 滤波钽电容爆裂。

**根因**: 钽电容额定电压 16V，工作电压 12V(降额 25%)，看似足够但热插拔瞬间产生大瞬变电流，击穿钽介质。

**解决**: 更换为铝电解电容。

**教训**: 钽电容耐瞬变电流能力弱，在有热插拔或大电流冲击的场合需降额 70% 以上使用，或改用铝电解电容。

#### 【案例 2-6】电容问题导致 CPU 工作不稳定

**现象**: 大量数据处理时偶发丢包。

**PLL 基础知识**:

锁相环(Phase-Locked Loop, PLL)是高速芯片内部的核心时钟生成模块。其基本结构:

鉴相器(PD) $(V_e)$
 环路滤波器(LF) $(V_c)$
 压控振荡器(VCO) $(f_{out})$ 反馈到鉴相器

说明: $(V_e)$
 = 误差电压, $(V_c)$
 = 控制电压, $(f_{out})$ = 输出时钟频率

PLL 的 VCO 是一个**对电源噪声极其敏感的模拟电路**。电源上的纹波和噪声会直接调制 VCO 的控制电压，导致输出时钟频率产生周期性抖动(jitter)。

**噪声 $\rightarrow$ 抖动的传递路径**:

VCC_PLL 噪声 → VCO 控制电压波动 → 时钟边沿偏移 → 建立/保持时间违例 → 数据采样错误

CPU 频率越高，对时钟抖动的容忍度越低。1GHz 时钟周期仅 1ns，1/4 周期抖动(250ps)就足以让 DDR 接口采样出错。

**根因**: CPU 锁相环供电 VCC_PLL 引脚仅有一个 10μF/1206 陶瓷电容，测量得纹波 50mV(含噪声 70mV)，导致 PLL 输出时钟边沿偏移约 1/4 周期。

**解决**: 靠近 VCC_PLL 引脚增加两个 0612 封装的 2.2μF 陶瓷电容(ESL 极小，提供高频低阻抗)和一个 0402 封装的 0.1μF 陶瓷电容，使纹波降至 10mV 以下。

**教训**: PLL 等敏感模拟电源引脚必须用多颗不同容值、不同封装的电容组合做低噪声滤波，且必须就近放置。

### 电容的三大作用

| 作用 | 原理 | 应用场景 |
|------|------|----------|
| **电荷缓冲池** | $\Delta V = \Delta Q / C $ ，电容吸收/释放电荷稳定电压 | 器件电源引脚去耦 |
| **高频噪声泄放** | $Z = 1/(2\pi f C) $ ，高频时电容呈低阻抗到地 | 电源滤波、旁路 |
| **交流耦合** | 通交流阻直流，隔离两侧直流偏置 | 高速差分信号互连 |

### 电容等效电路

![图2.8 电容的等效电路](images/Ch02_fig2.8_cap_equiv.png)

实际电容器件并非理想电容，包含寄生参数:

- ** $ C$
 (电容分量)**: 主导低频特性。理想情况下阻抗 $Z = 1/(2\pi f C) $ ，随频率升高而下降
- ** $ ESL$ (等效串联电感)**: 由引脚和电极结构引入，取决于**封装**(封装越小 ESL 越小)。主导高频特性
- ** $ ESR$ (等效串联电阻)**: 由介质损耗、电极电阻等组成。取决于**温度**和**频率**
- ** $ R_{leak}$ (并联泄漏电阻)**: 反映电容的漏电流特性，一般很大(>MΩ)，可忽略

**阻抗-频率曲线**(浴盆曲线)的三个区域:

| 频段 | 主导因素 | 阻抗变化趋势 |
|------|---------|------------|
| 低频段( $ f < f_{res} $ ) | **容性**(C) | $Z \propto 1/f $ ，随频率升高而下降 |
| 谐振点( $ f = f_{res} $ ) | **C = ESL** | 阻抗最小，仅由 ESR 决定 |
| 高频段( $ f > f_{res} $ ) | **感性**(ESL) | $Z \propto f $ ，随频率升高而上升 |

谐振频率公式: $f_{res} = 1 / (2\pi \sqrt{ESL \times C})$

> **核心结论**: 电容的滤波效果在谐振点附近最好(阻抗最低)。不同容值/封装的电容谐振点不同，组合使用可覆盖更宽频带。

### 交流耦合电容详解

#### 【案例 2-7】交流耦合电容选择不当引起数据帧出错

![图2.9 主控板与业务板之间的交流耦合电路](images/Ch02_fig2.9_AC_couple.png)

**现象**: 800Mbps LVPECL 差分信号采用 0.01μF 耦合电容，发送长连 0/连 1 码型时接收端出错。

**根因**: 耦合电容太小，无法维持长连 0/连 1 期间的充放电平衡。最小容值计算公式:

$C_{min} = 7.8 \times NUM \times T_b / R$

其中 $T_b = 1.25ns$
 (每比特), $NUM = 86$
 (最长连 0/连 1 长度), $R = 50\Omega$ (负载):

$C_{min} = 7.8 \times 86 \times 1.25 \times 10^{-9} / 50 = 16.8nF$

0.01μF(10nF) < 16.8nF，不满足要求，导致信号基线漂移，接收端判决错误。

**解决**: 改用 0.1μF 耦合电容。

**教训**: 交流耦合电容不能太小(否则长连 0/1 时信号失真)，也不能太大(影响边沿斜率)。高速设计中一般取 0.1μF。

### ESR 对电路的影响

ESR 在电路中有三个重要影响:

#### 1. 信号衰减

交流耦合电容串联在高速信号路径上时，ESR 相当于在信号中间串了一个小电阻。对于 0.1μF 陶瓷电容，ESR ≈ 10mΩ 可忽略;但对于铝电解电容(ESR 可达 Ω 级)，高频信号会因此产生明显衰减。

#### 2. 功耗与发热

根据电容损耗角正切值 $tan\delta = ESR / X_C $ ，较大的 ESR 会产生较大的损耗功率:

$P_{loss} = I^2 \times ESR$

当单板上同类型电容器件数目较多时，功耗预算就不得不考虑电容上的损耗。

#### 3. LDO 补偿(反常识)

并非所有场合 ESR 都越小越好。案例 2-9(LT1963 LDO)要求输出电容 ESR 不低于 20mΩ 才能保证环路稳定——ESR 太小反而会导致 LDO 振荡。设计时需要查阅器件手册对 ESR 的要求。

### 阻抗-频率特性详解

#### 阻抗随频率变化的原因

电容的阻抗公式: $Z = \sqrt{ESR^2 + (2\pi f L - 1/(2\pi f C))^2}$

- 低频( $ f \ll f_{res} $
 ): $2\pi f L$
 很小忽略, $1/(2\pi f C)$ 很大 → 呈**容性**
- 谐振点( $ f = f_{res} $
 ): $2\pi f L = 1/(2\pi f C) $ ，互相抵消 → 仅剩 **ESR**
- 高频( $ f \gg f_{res} $
 ): $1/(2\pi f C)$
 很小忽略, $2\pi f L$ 占主导 → 呈**感性**

#### 封装对阻抗的影响

ESL 的大小直接由封装决定:

| 封装 | ESL | 特点 |
|------|------|------|
| 0201 | 400pH | 极小封装，适合高密度 PCB |
| 0402 | 550pH | 常用高频去耦封装 |
| 0603 | 700pH | 通用封装，平衡尺寸与性能 |
| 0805 | 800pH | 中等尺寸，适合中等频率 |
| 1206 | 1250pH | 较大，谐振频率低 |
| 0612 | 63pH | 倒置结构，ESL 极低 ***表2.2《高速电路设计实践》

> **同一容值下，封装越小(ESL 越小)，谐振频率越高，高频滤波效果越好。**

#### 电容手册参数解读

实际电容数据手册中需要关注的参数:

- **标称容值**: 25°C、0V 下的值。实际有效值随温度和偏置电压变化
- **额定电压**: 电容能长期承受的最大直流电压
- **ESR**: 一般给出特定频率(如 100kHz)下的最大值
- **阻抗-频率曲线**: 最直观的滤波性能评估依据
- **温度特性**: X7R、X5R、Y5V 等代号表示温度稳定性
- **DC Bias 特性**: 陶瓷电容的容值随 DC 偏置电压升高而下降(X7R 在额定电压下可能下降 50% 以上)

#### 多电容并联: 展宽低阻抗频带

![阻抗-频率特性曲线(多电容对比)](images/Ch02_fig_impedance_curve.png)

上图对比了不同容值和封装组合的阻抗曲线。关键结论:

**有效的方法: 不同容值 + 不同封装**

例如 1μF/0603 + 0.01μF/0402:
- 1μF/0603 的谐振点约为 5.6MHz，负责中低频段
- 0.01μF/0402 的谐振点约为 80MHz，负责高频段
- 两者并联后，低阻抗频带从两个孤立谐振点扩展成一个连续的宽带

**无效的方法: 只换容值不换封装**

例如 1μF/0603 + 0.01μF/0603:
- 两者 ESL 相同(同为 0603 封装)，仅容值不同
- 小容值 0.01μF 的阻抗曲线被大容值 1μF 的曲线完全包含
- 并联后几乎没有展宽效果，浪费 PCB 面积和成本

**同型号并联: 只能降低谐振点阻抗**

![同型号电容并联(降低阻抗)](images/Ch02_fig_parallel_same.png)

两颗相同的电容并联，谐振点阻抗降为单颗的一半，但低阻抗频带宽度不变。

> **并联的目的是展宽低阻抗频带，这需要不同容值+不同封装的组合。仅并联同型号电容只能降低谐振点阻抗。**

#### 实际案例: 1μF + 0.01μF 的教训

某单板使用 0603 封装的 1μF 和 0.01μF 陶瓷电容并联，为 CPU 片内锁相环的电源滤波，设计者本意是利用两种不同容值展宽低阻抗频带。

但分析和测试表明: 两者封装相同(都是 0603)，ESL 相同(约 0.8nH)，0.01μF 的阻抗曲线完全被 1μF 的曲线包络，并联后阻抗曲线几乎没有变化。

**改进**: 将 0.01μF 改为 0402 封装(ESL 约 0.4nH)，两者的谐振点真正错开，低阻抗频带得以展宽。

### 常用电容类型详解

#### 1. 陶瓷电容 (MLCC)

**结构与特点**

陶瓷电容以陶瓷作为介质，两面喷涂金属作为极板。优点是体积小、价格低、ESR 小、高频特性好;缺点是容量小，且受到 DC Bias 效应影响。

**命名规范**

陶瓷电容的温度特性通过 EIA 编码标识，格式为"字母+数字+字母":

![表2.4 陶瓷电容符号表示法](images/Ch02_tab2.4_ceramic_code.png)

| 代号 | 含义 | 说明 |
|------|------|------|
| **第一个字母** | 最低工作温度 | X = -55°C, Y = -30°C, Z = +10°C |
| **中间数字** | 最高工作温度 | 4 = +65°C, 5 = +85°C, 6 = +105°C, 7 = +125°C |
| **最后一个字母** | 容值变化范围 | R = ±15%, V = +22%/-82%, P = ±10% |

**常用类型**:

| 类型 | 温度范围 | 稳定性 | 应用建议 |
|------|---------|--------|---------|
| **NPO/C0G** | -55°C~+125°C | ±30ppm/°C (极好) | 振荡/时序、精度要求高 |
| **X7R** | -55°C~+125°C | ±15% | 通用去耦、电源滤波(推荐) |
| **X5R** | -55°C~+85°C | ±15% | 常温去耦(消费电子推荐) |
| **Y5V** | -30°C~+85°C | +22%/-82% (极差) | ❌ 高速电路中避免使用 |

**DC Bias 效应**

陶瓷电容(尤其是 X7R、X5R、Y5V)的容值随施加的直流偏置电压升高而显著下降。X7R 在额定电压下有效容值可能降至标称值的 50%~70%。

![陶瓷电容特性曲线](images/Ch02_fig_ceramic_char.png)

> 标称 10μF 的 X7R 在 5V 偏置下可能只有 5~6μF 有效容值，设计时必须考虑。

**降额指南**: 电压降额至额定电压的 60%~80%，有效容值按标称值的 50%~70% 估算，避免使用 Y5V。

**ESR 特点**: 陶瓷电容 ESR 一般较小(几~几十 mΩ)，高频滤波优势。但容量小，不适合电源主滤波，需搭配钽电容或铝电解电容。

---

#### 2. 钽电容 (Tantalum Capacitor)

**结构与特点**

以金属钽作为阳极，表面形成致密的氧化钽(Ta2O5)介质层。电解质为二氧化锰或导电聚合物。

优点: 温度稳定性好、ESL 小(表贴封装)、体积小容量大;缺点: 耐冲击/浪涌能力弱。

**案例 2-12: 钽电容在 IO 接口电源滤波中的选型计算**

FPGA 的 IO 接口电源需滤波，要求滤波电容阻抗小于 IO 接口的动态阻抗。

**动态阻抗**: $Z_{dyn} = V_{ripple} / I_{ripple}$

- 电源 1.8V，允许纹波 5%: $V_{ripple} = 1.8V \times 5\% = 0.09V$
- 最大电流 0.5A，电流纹波 50%: $I_{ripple} = 0.25A$
- $Z_{dyn} = 0.09V / 0.25A = 0.36\Omega$

选 22μF 钽电容(AVX TPSB226x006)，查手册 100kHz 时 ESR 最大 0.6Ω:

$Z_{cap} = \sqrt{ESR^2 + (1/(2\pi f C))^2} = \sqrt{0.6^2 + (1/(2\pi \times 100kHz \times 22\mu F))^2} = 0.604\Omega$

0.604Ω > 0.36Ω，单颗不满足。**两颗并联**: $0.604\Omega / 2 = 0.302\Omega < 0.36\Omega $ ，满足。高频段搭配 0402/1μF/X5R 陶瓷电容。

> 钽电容选型关键: 滤波电容阻抗必须小于负载动态阻抗，必要时并联多颗。

**降额**: 普通应用降额 50%，热插拔/大冲击降额 70%以上。低阻抗电源(如 DC/DC)需串联小电阻限浪涌或改用铝电解。

---

#### 3. 铝电解电容 (Aluminum Electrolytic)

以电解液作为介质，铝壳为负极，内部金属板为正极。容量大(数百~数万 μF)、耐压高(数百 V)，但稳定性差:

- **ESR**: 大(几十 mΩ~几 Ω)，低温下急剧增大(案例 2-4)
- **温度**: 差，-30°C ESR 比常温大数倍
- **寿命**: 电解液逐渐干涸，容量下降、ESR 增大

**适用**: 低频滤波、缓启电路(如 -48V 缓启，降额至少 20%)、大能量存储。高速电路尽量避免。

---

#### 4. OSCON 电容

SANYO(现 Panasonic)导电聚合物电容，外形似铝电解但内部用聚合物代替电解液:

| 优点 | 缺点 |
|------|------|
| ESR 极小(几~几十 mΩ) | 多数为插装式 |
| 温度稳定性好 | 体积较大 |
| 价格低于钽电容 | 高频不如陶瓷 |
| 无电解液干涸 | — |

**适用**: DC/DC 输出滤波、替代钽/铝的中低频滤波。ESR 和温度特性介于钽和铝电解之间，性价比之选。

---

#### 5. 四种电容对比总结

| 类型 | ESR | 温度 | 高频 | 大容量 | 耐压 | 成本 | 推荐场景 |
|------|:---:|:----:|:---:|:-----:|:---:|:----:|---------|
| **陶瓷** | 极低 | 好 | ✅✅ | ❌ | 中低 | 低 | 高频去耦、PLL |
| **钽** | 低 | 好 | ✅ | ✅ | 低 | 高 | CPU/IO 滤波 |
| **铝电解** | 高 | 差 | ❌ | ✅✅ | ✅✅ | 低 | 低频、缓启 |
| **OSCON** | 很低 | 较好 | ✅ | ✅ | 中 | 中 | DC/DC 替代 |

> 实际设计是**多种电容组合**: 钽/铝电解负责中低频，陶瓷负责高频，没有一种电容能覆盖全频段。

#### 【案例 2-11】陶瓷电容选型错误导致丢包

**现象**: 降成本后将交换芯片电源的 10μF 滤波电容从 X7R 改为 Y5V，高温 55°C 下丢包。

**根因**: Y5V 电容在 80°C 时有效容值降至标称值的 40% 以下(仅 3.78μF)。

**解决**: 换回 X7R 类型。

**教训**: Y5V 电容温度稳定性极差，高速电路中应选用 NPO、X7R、X5R，避免使用 Y5V。

> 钽电容需特别注意: 感性负载/小串联电阻/大瞬变电流场合，额定电压需降额 70% 以上。

### 去耦电容与旁路电容

这两个概念在工程中经常混用，但设计目的和放置位置有本质区别:

| | 去耦电容 (Decoupling) | 旁路电容 (Bypass) |
|------|----------------------|-------------------|
| **目的** | 为本器件提供本地电荷池，防止器件自身产生的电流瞬变干扰其他电路 | 为前级/外部的噪声提供一条低阻抗到地的通路，防止噪声进入本器件 |
| **视角** | 从芯片往外看(防止干扰别人) | 从芯片往里看(防止别人干扰自己) |
| **放置** | 紧贴器件 Vcc 引脚与 GND 之间 | 放置在噪声源与受扰器件之间的线路上 |
| **典型值** | 0.1μF + 1μF (Vcc 引脚处) | 0.1μF/0402 或磁珠+电容组合 |

**去耦电容的工作机制**:

器件工作时需要瞬间大电流(如输出电平切换时)，如果完全依赖电源模块提供该瞬态电流，会因为走线电感的存在产生较大的压降和噪声。去耦电容作为"本地电荷池"就近存储在电荷，在器件需要瞬态电流时快速供应，减小电压跌落:

$\Delta V = \frac{\Delta I}{\Delta t} \times L_{trace} + \frac{\Delta Q}{C_{decoup}}$

**去耦电容的选型原则**:

1. **就近放置**: 距离 Vcc 引脚越近越好(缩短高频回路)，0402 封装贴得越近 ESL 越小
2. **多值并联**: 1μF/0603 + 0.1μF/0402 + 0.01μF/0402 的组合覆盖宽频带
3. **每个 Vcc 引脚至少要配一颗**: 高速芯片的每个 Vcc 引脚都应该有就近的去耦电容
4. **低频去耦靠大电容**: 10~100μF 钽电容放在 PCB 板级，负担低频瞬态

**工程误区**:

- ❌ "去耦电容越多越好" → 过多相同容值的电容并联效果有限(谐振点不偏移)，浪费空间
- ❌ "0.1μF 通吃所有频率" → 0.1μF/0603 的谐振点约 17MHz，在 GHz 级已呈感性
- ❌ "距离远一点也没关系" → 每增加 1mm 走线约增加 1nH 电感，谐振频率大幅降低

**实际案例**: 某 FPGA 设计中，VCC_1V0 电源轨用了 8 颗 0.1μF/0603 电容(全部相同)。测试发现 50MHz 处仍有较大阻抗峰。分析: 8 颗相同电容谐振点相同(约 17MHz)，仅降低了该点的阻抗，但 50MHz 处已脱离容性区。**改进**: 改为 4 颗 1μF/0603 + 2 颗 0.1μF/0603 + 2 颗 0.01μF/0402，50MHz 阻抗下降 60%。

> **总结**: 去耦靠的是"不同谐振点的多颗电容"+"就近放置"，不是靠数量堆砌。

## 2.3 电感的选型及应用

### 经典案例

#### 【案例 2-13】LC 低通滤波导致输出电源纹波偏大

![图2.22 PHY芯片电源的产生电路](images/Ch02_fig2.22_PHY_power.png)

**现象**: PHY 芯片外部晶体管产生的 1.5V 电源存在 160mV 纹波。

**根因**: LC 滤波器(L4 + C1225)谐振频率 1.59MHz，恰好处于晶体管工作频段(1~100MHz)内，发生谐振。实际上该电路本身就是 LDO，无需 LC 滤波。

**解决**: 删除 L4，晶体管集电极直连输出。

**教训**: 不要迷信 LC 滤波，先搞清楚电路结构再决定是否需要。

#### 【案例 2-14】大电流通路 PI 型滤波造成电压衰减

![图2.23 PI型滤波电路及PCB电源平面分配](images/Ch02_fig2.23_PI_filter.png)

上图左侧为 MAC 芯片 PI 型滤波电路(MAC1/MAC2 各一串 L+双电容)，右侧为 PCB 内层 1.8V 电源平面布置图，展示电源模块经 550mil 宽路径供电至 MAC 芯片的走线布局。

**现象**: MAC 芯片 1.8V 电源 P1V8_MAC1(1.67V)和 P1V8_MAC2(1.61V) 低于最低要求 1.62V。

**根因**: 四重电压衰减叠加:
1. 电源模块输出过孔仅一个，通流不足
2. 电源平面路径上被信号过孔阻断，有效宽度从 550mil 缩至 180mil
3. B→D 路径有效宽度仅 95mil
4. 电感 L3/L5 直流电阻造成 0.1V 压降

**解决**: 花焊盘过孔 + 新增过孔 + 其他层增加电源平面 + 低直流电阻电感。

**教训**: 电源平面设计需考虑过孔通流、路径有效宽度、信号过孔的阻断效应等因素，且经验公式结果需降额使用。电源滤波用电感上的压降不可忽视。

---

### 电感在高速电路中的三大作用


#### 1. 通直流、阻交流

$Z = j\omega L = 2\pi f L $
 ，频率越高感抗越大。直流分量( $ f=0 $ )感抗为零，交流分量频率越高越难通过。此特性用于电源滤波: 串联电感阻隔电源线上的高频噪声(交流)，让直流顺利通过。

#### 2. 阻碍电流变化，保持电流稳定

当流过电感的电流发生变化时，电感产生反向感应电动势。电流增大时产生反向电压阻碍增大;电流减小时产生正向电压补充减小。这一机制使电感在 DC/DC 电源中作为储能元件，在开关管导通时存储能量、关断时释放能量。

#### 3. 滤波 — 与电容构成低通滤波器

电感和电容构成的 LC 低通滤波器，对高频信号的衰减率为 $-40dB/decade $
 (二阶)，远优于单电容的 $-20dB/decade $ (一阶)。LC 滤波器的截止频率:

$f_c = 1 / (2\pi \sqrt{L \times C})$

> 电感在高速电路中主要用于电源滤波和 EMI 抑制，核心是利用其感抗的频率选择特性。

---

### 电感的主要参数

| 参数 | 符号 | 单位 | 说明 |
|------|------|------|------|
| **电感值** | $L$ | H (亨利) | 核心参数，频率 $f$ 下的感抗 $X_L = 2\pi f L$ |
| **自谐振频率** | $f_{SR}$ | Hz | 电感与寄生电容谐振的频率。**工作频率必须低于 $f_{SR} $ **，否则电感呈容性 |
| **直流电阻** | $DCR$ | $\Omega$ | 线圈自身的电阻。决定了**直流通路压降**( $ V_{drop} = I \times DCR $ )和**发热**( $ P = I^2 \times DCR $ ) |
| **额定电流** | $I_{sat}$ | A | 磁芯饱和或温升限制的最大电流。超过后电感值急剧下降 |
| **品质因数** | $Q$ | — | $Q = \omega L / R_{loss} = 2\pi f L / ESR $ 。反映了电感的能量存储效率 |

![电感特性曲线(阻抗与Q值)](images/Ch02_fig_inductor_char.png)

上图左: 实际电感在自谐振频率前呈感性( $ Z \propto f $
 )，谐振后呈容性( $ Z \propto 1/f $ )。右: 不同类型电感的 Q 值峰值所在频率不同，需按工作频率选型。

---

### 三种电感类型详解

#### 1. 高频信号用电感 (nH 级)

| 项目 | 内容 |
|------|------|
| **电感值范围** | 0.6~390 nH |
| **自谐振频率** | >1 GHz |
| **额定电流** | 几十~几百 mA |
| **典型结构** | 空芯绕线或薄膜式 |
| **主要材质** | 空气介质/陶瓷骨架 |

**应用特点**:
- Q 值高，在 GHz 频段仍有良好特性
- 用于 RF 射频电路的阻抗匹配、谐振、扼流
- **注意事项**: 空芯结构无磁饱和问题，但相邻器件的寄生耦合需注意

#### 2. 一般信号用电感 (μH 级)

| 项目 | 内容 |
|------|------|
| **电感值范围** | 0.01~1000 μH |
| **自谐振频率** | 几十~几百 MHz |
| **额定电流** | 几~几十 mA |
| **典型结构** | 叠层式或多层绕线 |
| **主要材质** | 铁氧体(ferrite)磁芯 |

**应用特点**:
- 用于板间高速信号滤波、抑制共模噪声
- 铁氧体磁芯高频损耗大，适合做 EMI 滤波
- **注意事项**: 铁氧体在大电流下可能磁饱和，饱和后电感值急剧下降。工作频率需低于 $f_{SR}$

#### 3. 电源用电感 (μH ~ mH 级)

| 项目 | 内容 |
|------|------|
| **电感值范围** | 1~470 μH (DC/DC 常用)，更大可达 mH 级 |
| **自谐振频率** | 几~几十 MHz |
| **额定电流** | 几十 mA~几 A (大功率可到几十 A) |
| **典型结构** | 绕线式(磁屏蔽或非屏蔽) |
| **主要材质** | 铁粉芯/铁硅铝/锰锌铁氧体 |

**应用特点**:
- DC/DC 转换器的核心储能元件
- 决定电源纹波和瞬态响应
- **DCR 是关键参数**: 大电流下 DCR 引起的压降和发热不可忽视
- **选型公式**: 纹波电流 $\Delta I_L = (V_{in} - V_{out}) \times D / (L \times f_{sw}) $
 ，其中 $D$
 为占空比， $ f_{sw}$ 为开关频率
- **注意事项**:
 - 额定电流 $I_{sat}$
 必须大于最大峰值电流( $ I_{peak} = I_{load} + \Delta I_L / 2 $ )
 - 磁屏蔽电感优于非屏蔽(减小 EMI)
 - DCR 越小越好，但 DCR 和尺寸/成本成反比

#### 三种电感总结

| 类型 | 核心关注 | 工作频率 | 典型应用 | 失效模式 |
|------|---------|---------|---------|---------|
| **高频信号(nH)** | Q 值、 $ f_{SR}$  | >1GHz | RF 匹配、谐振 | 寄生耦合 |
| **信号(μH)** | $f_{SR} $ 、磁饱和  | 几~几百 MHz | EMI 滤波、信号隔离 | 磁饱和 |
| **电源(μH~mH)** | DCR、 $ I_{sat}$  | <10 MHz | DC/DC 储能、电源滤波 | 磁饱和、过热 |

> 三种电感的本质都是线圈 + 磁芯，区别在于工作频段和功率等级。选型时的优先级: **电源电感先看 DCR 和电流 → 信号电感先看 SRF → 射频电感先看 Q 值**。

## 2.4 磁珠的选型及应用

### 磁珠的滤波机理

磁珠通过低频反射 + 高频吸收的方式滤除噪声，与电容(泄放)和电感(反射)的机理都不同:

| 器件 | 低频行为 | 高频行为 | 噪声去向 |
|------|---------|---------|---------|
| **电容** | 容性低阻抗 | 感性(谐振后) | 泄放到地(仍在系统中) |
| **电感** | 感性高阻抗 | 容性(谐振后) | 反射回源头(仍在系统中) |
| **磁珠** | 感性(反射) | **电阻性(吸收)** | 转化为热能(彻底消除) |

**转换点频率**: 磁珠总阻抗 $Z = \sqrt{R^2 + X^2} ，其中 R为电阻成分， X$ 为电抗成分。

$R = X$ 处的频率即为转换点:
- $f < f_{conv} : 电感性(  X > R  )，反射噪声$
- $f > f_{conv} : 电阻性(  R > X  )  ，吸收噪声转化为热能$

> 磁珠是耗能器件，不会产生自激;电感是储能器件，与电容配合可能产生 LC 自激振荡。

![磁珠阻抗-频率特性曲线](images/Ch02_fig_bead_char.png)

上图左: 磁$珠的  |Z|  、  R  、 X 随频率变化曲线，转换点处 R=X  。右: 磁珠与电感的阻抗$对比——在 >50MHz 时磁珠阻抗以电阻成分为主吸收噪声，而理想电感仍呈感性阻抗。

### 磁珠的主要参数

| 参数 | 说明 | 选型要点 |
|------|------|---------|
| **阻抗 @ 100MHz** | 磁珠最常用的标称参数，如 120Ω@100MHz | 按目标噪声频率选择，通常在 30~1000Ω |
| **额定电流** | 磁珠能承受的最大直流电流 | **选型的关键**。超过后磁芯饱和，磁珠失效为导线 |
| **直流电阻(DCR)** | 磁珠的线圈电阻 | 电源路径上的 DCR 会产生压降( $ V = I \times DCR $ )，大电流时要核算 |
| **转换点频率** | $R=X$ 处的频率 | 决定滤波类型: 低于此频率为反射型，高于为吸收型 |

> 磁珠选型第一步看额定电流——工作电流必须 ≤ 额定电流的 60%~70%，否则磁芯饱和导致磁珠失去吸收噪声的能力。

---

### 应用要点与注意事项

#### 电源滤波 vs 信号滤波

- **电源路径**: 磁珠常串联在电源入口或芯片电源引脚前，滤除电源线上的高频噪声。需注意 DCR 压降和额定电流
- **信号路径**: 磁珠串联在高速信号线上(如给时钟、复位信号供电的电源线)。信号频率不能落在磁珠的吸收频段内(否则信号被衰减)

#### 常见误区

1. **额定电流被超过**: 磁芯饱和 → 磁珠变为纯导线 → 失去滤波作用
2. **DCR 压降被忽略**: 电源磁珠上 $V_{drop} = I \times DCR $ ，大电流下可能掉电
3. **信号被磁珠吸收**: 磁珠的吸收频段覆盖了信号频率 → 信号幅度下降

> 磁珠选型三要素: 额定电流 > DCR 压降 > 阻抗频率特性

---

### 磁珠 vs 电感 — 对比总结

| 对比项 | 磁珠 | 电感 |
|--------|------|------|
| 滤波机理 | **吸收**噪声→热能 | **反射**噪声(谐振/反射) |
| >50MHz 高频性能 | ✅ 好(电阻性吸收) | ❌ 差(寄生电容致自谐振) |
| 抗 EMI | ✅ 好(吸收不辐射) | ❌ 可能辐射干扰 |
| 自激风险 | ❌ 不会(耗能器件) | ⚠️ 可能(储能器件，与C形成LC) |
| 额定电流 | 较小(受磁芯饱和限制) | 较大(可做到几十A) |
| 直流电阻 | 一般较小(几十~几百 mΩ) | 较大(几百 mΩ~几 Ω) |
| 等效模型 | R + L 串联(频率相关) | L + C_par 并联 |

**共同点**:
- 工作电流不可超过额定值
- 直流电阻造成压降，大电流下需核算
- 选型需参考频率特性曲线

### 【案例 2-15】误用磁珠造成过流保护电路失效

![图2.32 带电流监控功能的DC/DC电源电路](images/Ch02_fig2.32_overcurrent.png)

**现象**: 即使将电流检测电阻从 0.025Ω 改为 0.1Ω，也无法触发过流保护。

**根因**: 10V 电源进入单板后分为两路: 一路经电阻到 SENSE 引脚(不耗流)，一路经磁珠后产生 Vout。实际被监控的是不耗流的路径，真正的大电流路径(经磁珠)未被监控，即使负载过流，SENSE 引脚检测到的电流仍然很小。

**解决**: 去掉磁珠，使过流检测路径与电源输出路径一致。

**教训**: 磁珠串入电源路径后，其后的电流不会被电源芯片的电流监控电路检测到。不要盲目在电源路径串联磁珠。

### 选型速查

| 应用场景 | 推荐磁珠 | 注意事项 |
|---------|---------|---------|
| 数字芯片电源去耦 | 30~100Ω @100MHz | 核算额定电流是否大于实际电流 |
| 模拟电源(PLL/ADC) | 100~600Ω @100MHz | 串联磁珠 + 对地电容构成 π 型滤波 |
| 时钟/复位信号电源 | 30~60Ω @100MHz | 不能落在信号基频附近 |
| I/O 接口电源 | 60~120Ω @100MHz | 注意 DCR 压降 |
| DC/DC 输出 | ❌ 不建议用磁珠 | 改用低 DCR 电感

## 本章小结

| 器件 | 选型核心 | 关键参数 |
|------|---------|---------|
| 电阻 | 阻值 + 额定功率 + 精度 + 尺寸 | 四选型维度 |
| 电容 | 容值 + 封装(ESL) + 类型(温度特性) + ESR | 阻抗-频率曲线 |
| 电感 | 电感值 + 自谐振频率 + 直流电阻 + 额定电流 | 频率特性曲线 |
| 磁珠 | 转换点频率 + 额定电流 + 直流电阻 + 谐振频率 | 转换点 = R=X 处频率 |
